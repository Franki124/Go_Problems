Imagine you are a Go developer at a time travel agency and need to create a program that schedules time travel trips for clients. Each trip has three time points: departure date, historical event date, and return date. However, the program must validate if the trip is valid based on the following rules:

  • The historicalEventDate must be in the past relative to the departureDate.

  • The returnDate must be in the future relative to the departureDate.

  • All travels are only to the past!

Your task is to write the additional required code for the readDate() function to create a Date object from the input dates, and the code for the conditional statements to check if the trip is valid.

If the trip is valid, your program should print "Valid trip!" and if the trip is not valid it should print "Invalid trip!".


Sample Input:
2023 01 01
1969 07 20
2023 01 02

Sample Output:
Valid trip!


Sample Input:
2023 01 01
2024 07 20
2023 01 02

Sample Output:
Invalid trip!



Memory limit: 256 MB
Time limit: 8 seconds
